Text Simplification In Practice Evaluation
Instructions: Your task in this evaluation is split into three phases. After reading carefully these guidelines, start the first phase by pressing the Next button. Please perform all tasks as quickly and as accurately as possible. Estimated time is 2 minutes per one text.
  1. Reading: Read the text carefully and press Next only after you finished reading.
  2. Comprehension: Answer the questions based on the text. You can consult the text while answering them.
  3. Evaluation: Let us know how confident you are in the answers and provide some more details about the questions we asked.
The Next button will be unlocked only after you answered all questions. There is no way to go back to previously finished segments. Use the following guidelines in the evaluation phase (0 - least, 5 - most):
  • Confidence: 0 - my answers are random, 5 - all my answers are very likely correct.
  • Answerability: 0 - the text did not help in answering any of the questions, 5 - the text covered directly all the answers.
  • Necessariness: 0 - the text provided needless details that did not help in understanding the text, 5 - the text was focused on the main points.
  • Complexity: 0 - the text was difficult to read and I would be able to mark specific parts which need simplifying, 5 - the text was easy to read and it is not possible to simplify it further.
  • Grammaticality: 0 - the text contains some grammar/fluency/continuity errors, 5 - the text is very grammatical, likely written by a human.
Only if you are very unsure about an answer, select I don't know.
